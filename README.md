# 5G-SIMULATION-SYSTEM
Este proyecto implementa una red móvil 5G Standalone (SA) completamente funcional sobre un único equipo, integrando los tres dominios de una red celular moderna:  Core Network: Open5GS  ,   Radio Access Network: srsRAN Project    gNB (Next Generation NodeB)  . Air Interface: SDR USRP B200 radiando señal real hacia un teléfono comercial
